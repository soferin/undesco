📡 SubscribeToChat (STREAM)

Socket: chat.subscribe_to_chat

REQUEST

Campo Tipo Descripción
chat_id string ID del chat
user_id string ID usuario 
{
  "chat_id": "chat123",
  "user_id": "user1"
}

STREAM RESPONSE (ChatEvent)

Tipo de evento Descripción
new_message Nuevo mensaje en tiempo real
user_typing Usuario escribiendo
user_stopped_typing Usuario dejó de escribir
user_joined Usuario entró al chat
user_left Usuario salió del chat
message_edited Mensaje editado
message_deleted Mensaje eliminado

EVENT EXAMPLE (new_message)

{
  "new_message": {
    "id": "msg1",
    "chat_id": "chat123",
    "sender": {
      "id": "user1",
      "name": "Juan"
    },
    "message_type": "TEXT",
    "message": "hola en vivo",
    "ciphertext": "",
    "source": "",
    "created_at": "2026-04-13T10:00:00Z",
    "updated_at": "2026-04-13T10:00:00Z",
    "response_msg_id": "",
    "deleted_by": [],
    "keys": [],
    "conversation_id": "conv1"
  }
}
